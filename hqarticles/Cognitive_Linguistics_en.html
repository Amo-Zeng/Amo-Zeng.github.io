Dynamic Coupling Cognitive Linguistics (DCCL): A Paradigm Revolution in Cognitive Linguistics under a Unified Framework

— Toward a New Paradigm of Embodied, Enactive, and Technologically Embedded Cognitive Science

I. Core Question: Is Language a Static Mirror of Cognition, or a Dynamic Selector and Resonant Interface in a Triadic Cognitive-Environmental-Technological Coupling System?

Since its foundation in the 1980s on the theoretical pillars of embodied cognition and conceptual metaphor, cognitive linguistics has been trapped in a fundamental theoretical tension: Is language merely a passive symbolic output of cognitive structure, or an active generative interface that participates in cognitive construction? The traditional paradigm—characterized by the triad of unidirectional causality, individualism, and environment-as-container—treats language as a "window" or "shadow" of the mind, attributing its variation to individual differences or random drift in cultural transmission. However, this paradigm has suffered systematic failure when attempting to explain the co-evolutionary phenomena between language and cognition in contemporary socio-technical ecosystems.

Notably, three key studies collectively reveal a more disruptive proposition:
In complex socio-technical ecosystems, language is not merely a passive reflection of cognition, but actively shapes, reconstructs, and evolves cognitive structures through dynamic coupling among cognition, language, environment, and technology.

This proposition forms the theoretical starting point of DCCL and exposes a deep paradigmatic conflict within traditional cognitive linguistics:

  • TCD/TCM studies (language as a source of cognitive selection pressure) show that linguistic structures (e.g., passive voice, emotional expressions) form frequency-dependent cognitive selection pressure (CSP) through positive feedback loops involving usage frequency, social transmission, and algorithmic feedback, driving phase transitions in cognitive styles at the population level (e.g., externalization of responsibility, emotional attribution). Language thereby becomes a niche constructor.
  • PRH studies (Predictive Resonance Hypothesis) challenge the neurobiological assumption of static symbolic representation, proposing that the neural basis of language comprehension is a cross-modal predictive resonance process via neural oscillations, where semantic meaning emerges transiently through phase synchronization, not stored as static representations in brain regions. Language is thus a neurodynamic oscillation protocol.
  • HMCE studies (Human-Machine Cognitive Co-evolution) further reveal that AI systems, algorithmic recommendations, and auto-completion tools have become new linguistic environments. Their feedback mechanisms not only select linguistic structures but also reshape human cognitive strategies (e.g., anthropomorphic attribution, reduced sense of control) through human-machine co-evolution.

Therefore, if language can selectively shape cognition (TCD), drive neural oscillations (PRH), and be regulated by technological co-evolution (HMCE), then cognitive linguistics must transcend the closed model of the "individual mind" and enter a new paradigm of systemic coupling, nonlinear evolution, and technological embeddedness. This is not merely theoretical expansion, but an ontological revolution: language is no longer the "shadow" of cognition, but the core interface and dynamic node of a Dynamic Coupling Cognitive System (DCCS).

II. Theoretical Integration: Dynamic Coupling Cognitive Linguistics (DCCL) — A Unified Framework

We propose Dynamic Coupling Cognitive Linguistics (DCCL), integrating TCD, PRH, and HMCE into a cross-scale, multimodal, and causally closed unified framework. The central claim of DCCL is:

Language is a cross-scale cognitive interface within the triadic cognitive-environmental-technological system, achieving selection, resonance, and co-evolution through the dynamic coupling of "use-oscillation-feedback".

2. Methodological Integration: Triple Coupling Validation Chain (TCVC)

(1) Cross-Scale Coupling Modeling: ABM + Neural Oscillation Simulator (TCD ↔ PRH ↔ HMCE)

  • Agent-Based Modeling (ABM): Agents possess cognitive, linguistic, neural, and technological attributes, embedded in social networks and algorithmic environments.
  • Neural Oscillation Model (NOM): Based on Kuramoto models, simulating phase synchronization in brain regions (e.g., A1-M1-ACC), driven by semantic predictions from LLMs.
  • Coupling mechanism: Language use frequency → neural plasticity (θ-coupling update) → behavioral output → algorithmic recommendation → language use frequency.
  • Metrics: Population linguistic entropy, phase transition thresholds, cross-scale coupling strength.
Enhanced logical rigor: Through cross-scale parameter mapping, TCVC enables causally closed modeling of social evolution (ABM), neurodynamics (NOM), and technological intervention (LLM), avoiding "black-box" explanations.

(2) Neuro-Behavior-Technology Co-Tracking: fNIRS-EEG + NLP + Algorithm Logs (PRH ↔ TCD ↔ HMCE)

  • Synchronous data collection: Record fNIRS-EEG (neural), NLP (linguistic), and algorithm logs (technological) during natural social media interactions.
  • Analytical methods:
    • Granger causality: Test whether linguistic changes precede neural oscillation changes (challenging "cognition → language" unidirectionality);
    • Vector Autoregression (VAR): Model dynamic coupling equations among language, neural, and algorithmic variables;
    • Mediation analysis: Test the pathway "algorithmic recommendation → language use → θ-coupling → cognitive output".
Key finding: Algorithm preference for emotional expressions → increased passive voice use → reduced θ-coupling → increased external attribution (the TCD-PRH-HMCE chain), empirically supporting the triadic causal coupling.

(3) Causal Interventions: tACS + Algorithmic Perturbation Experiments (PRH ↔ HMCE)

  • tACS intervention: Apply θ-band stimulation (6Hz) to Broca’s area; measure semantic integration speed, language use patterns, and algorithmic responses.
  • Algorithmic perturbation: A/B testing to adjust recommendation weights; measure changes in linguistic style and θ-coupling.
  • Predictions:
    • tACS synchronization → increased θ-coupling → increased active voice → reduced algorithmic exposure (neural → behavioral → technological);
    • Algorithm reduces emotional weight → reduced emotional expression → increased θ-coupling (technological → behavioral → neural).
Causal closure: Demonstrates bidirectional causality among neural, behavioral, and technological levels, completely overturning the traditional unidirectional model.

3. Application and Ethics: Cognitive Impact Assessment (CIA)

DCCL introduces the Cognitive Impact Assessment (CIA) framework, requiring all language-related technologies (AI, education, public policy) to undergo:

  • Selection pressure assessment: Does it promote de-agentification, emotionalization, or other cognitive strategies?
  • Resonance risk assessment: Does it impair θ-γ coupling, affecting semantic integration?
  • Co-evolution warning: Does it create algorithmic-cognitive feedback loops leading to cognitive style shifts?
Ethical principle: Language design must undergo neuroplasticity impact assessment to avoid unconsciously shaping harmful cognitive ecosystems. This marks a paradigm shift from "technological neutrality" to "cognitive responsibility".
