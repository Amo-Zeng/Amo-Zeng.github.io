Triple Upgrading of Value Engineering

Constructing the Theoretical Framework and Practical Pathway for Next-Gen VE (Next-Generation Value Engineering)

1. Introduction: The Need for a Paradigm Shift in Value Engineering

In the context of digital transformation, carbon neutrality goals, and global supply chain restructuring, the traditional Value Engineering (VE) model based on the static "function-cost" two-dimensional optimization framework is no longer sufficient to address the challenges of value creation in complex systems. Its limitations are primarily reflected in:

  • Static functional definitions that ignore dynamic user needs and system evolution;
  • Cost analysis limited to explicit inputs, overlooking hidden costs (e.g., carbon footprint, social impact);
  • Singular value creation pathways lacking cross-scale coordination mechanisms.

In this context, this paper proposes a theoretical framework for Next-Generation Value Engineering (Next-Gen VE) through Triple Upgrading — Functional Upgrading, Cost Upgrading, and Value Upgrading — to reconstruct the core logic of VE and form the Triple-Upgrading Synergy Model (TUSM). This transformation shifts VE from a "cost-reduction tool" to a strategic value engine.

2. Theoretical Reconstruction: The Academic Foundations of Triple Upgrading

2.1 Functional Upgrading: From Static Functions to Dynamic Value Streams

Traditional VE defines "function" as the performance of a product/service required by users, but fails to capture its contextuality, configurability, and evolvability. Drawing on Value Stream Engineering (VSE) and Service-Dominant Logic (S-D Logic), we propose the Dynamic Function Matrix (DFM):

$$DFM = \langle F_t, C_t, I_t, S_t \rangle$$

Where $F_t$ is the function set, $C_t$ represents contextual variables, $I_t$ is interaction intensity, and $S_t$ is system state. DFM enables real-time mapping of functional states via Digital Twin, realizing "Function as a Service" (FaaS).

Theoretical Basis: S-D Logic (Vargo & Lusch, 2004, 2016) emphasizes value co-creation among multiple actors, requiring functions to be embedded in service systems; Value Stream Theory (Womack & Jones, 2003) provides a process-oriented perspective.

2.2 Cost Upgrading: From Explicit Costs to Full Life-Cycle Social Costs

Traditional cost analysis focuses on manufacturing and procurement, ignoring hidden costs and externalities. We propose the Life-Cycle Social Cost (LCSC) Model, integrating:

  • Economic Costs (CAPEX/OPEX)
  • Environmental Costs (carbon footprint, resource use, per ISO 14040)
  • Social Costs (labor safety, community impact, per GRI standards)
  • Risk Costs (supply chain disruption, policy compliance)

LCSC uses a Cost-Impact Matrix (CIM) for quantification and supports Multi-Criteria Decision Making (MCDM).

Theoretical Basis: Life-Cycle Assessment (LCA, Guinée et al., 2002), Sustainable Development Accounting (Bebbington & Larrinaga, 2014), and Externality Theory (Pigou, 1920).

2.3 Value Upgrading: From Economic Value to Multi-Stakeholder Value Co-Creation

Traditional VE centers on "cost-performance ratio," neglecting the subjectivity and diversity of value. We introduce Stakeholder Value Co-Creation (SVCC) theory and propose the Multi-Dimensional Value Framework (MDVF), encompassing:

  • Economic Value (enterprise, customer)
  • Environmental Value (ecology, community)
  • Social Value (equity, inclusivity)
  • Innovative Value (knowledge spillover, technology diffusion)

MDVF employs a Dynamic Value Weighting Algorithm, combining AHP and Fuzzy Comprehensive Evaluation, to quantify and prioritize values.

Theoretical Basis: Stakeholder Theory (Freeman, 1984), Shared Value (Porter & Kramer, 2011), and ESG Framework (Eccles et al., 2014).

3. Framework Integration: The Triple-Upgrading Synergy Model (TUSM)

3.1 Model Structure

TUSM is a spiral co-evolutionary system (see Figure 1), with the following core mechanisms:

  1. Functional Upgrading drives Cost Upgrading: Dynamic functional requirements force cost structure reconfiguration (e.g., modular design reduces maintenance costs);
  2. Cost Upgrading feeds back to Value Upgrading: LCSC identifies high-cost/high-impact nodes, guiding value redistribution;
  3. Value Upgrading reinforces Functional Upgrading: Stakeholder value preferences reshape functional definitions (e.g., user preference for green products).

Synergy Mechanism: Real-time linkage across the three dimensions is enabled via a Digital Twin platform, forming a closed-loop "sense-analyze-decide-feedback" cycle.

4.2 Case Validation: New Energy Vehicle (NEV) Battery System

  • Functional Upgrading: Real-time prediction of battery State of Health (SOH) via Digital Twin; function evolves from "energy storage" to "energy service node";
  • Cost Upgrading: LCSC identifies recycling costs as 35% of life-cycle cost, driving design for reuse;
  • Value Upgrading: MDVF shows user weight of "carbon label" at 0.28, driving green certification investment;
  • Synergistic Optimization: TUSM recommends "modular + reuse + carbon label" combination, achieving 22% total value increase and 14% cost reduction.

5. Strategic Implications and Future Outlook

5.1 Strategic Elevation: From Tool to Strategic Engine

Next-Gen VE is no longer merely a "cost-reduction tool," but a core engine for achieving the following strategic goals:

  • Sustainable Competitive Advantage: Building differentiated barriers through value co-creation;
  • ESG Compliance & Brand Premium: LCSC and MDVF support ESG reporting and green financing;
  • Resilient Supply Chains: Function-cost-value synergy enhances risk resilience;
  • Innovative Ecosystems: Digital Twin and MAS enable open innovation.

Strategic Positioning: VE should be embedded in strategic management processes (e.g., Balanced Scorecard, Strategy Map) as a core methodology of the "Value Strategy Committee."

5.2 Future Outlook: Four Key Trends

  1. AI-Driven Autonomous Value Engineering: TUSM models powered by Reinforcement Learning (RL) will enable "self-optimization," reducing human intervention;
  2. Value Tokenization via Blockchain: Smart contracts will automate value distribution, realizing "value as currency";
  3. Cross-Organizational Value Synergy Networks: Federated learning-based distributed TUSM will support supply chain/industrial cluster collaboration;
  4. Value Engineering Metaverse: Virtual pre-simulation of value solutions will enable "zero-cost trial and error."

Policy Implication: We recommend integrating Next-Gen VE into national standards for smart manufacturing and green manufacturing (e.g., ISO 56000 series) to promote industrial-scale adoption.

6. Conclusion

This paper proposes the Triple-Upgrading Synergy Model (TUSM) for Next-Generation Value Engineering (Next-Gen VE), reconstructing the VE theoretical framework through Functional Upgrading (Dynamic Function Matrix), Cost Upgrading (Life-Cycle Social Cost), and Value Upgrading (Multi-Dimensional Value Co-Creation). TUSM not only resolves the limitations of traditional VE — static, partial, and singular — but also elevates VE into a strategic-level value creation system. Empirical case studies validate its feasibility and effectiveness.

In the future, Next-Gen VE, driven by AI, blockchain, and the metaverse, will evolve toward autonomy, networkization, and virtualization, becoming a core methodology for advancing corporate sustainability and industrial value leapfrogging.
