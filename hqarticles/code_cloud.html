    AI对数据隐私的挑战

    集中式AI平台通过垄断用户数据形成市场壁垒，加剧隐私侵害风险。

    核心风险点

    • 数据寡头化：头部平台控制90%数据资源
    • 算法黑箱：强化垄断者的信息优势
    • 隐私悖论：用户无法自主选择退出

    监管与技术的双重路径

    需通过**GDPR数据可移植权** + **区块链存证技术**，构建透明化数据市场。

    案例：欧盟DMA法案如何限制AI垄断？

    M DMA要求大型平台开放API，允许第三方接入AI服务（如推荐系统），打破数据垄断。

    用户赋能工具

    • 数据钱包：允许用户选择性授权数据使用
    • 差分隐私：在数据集中添加噪声，防止个体识别
