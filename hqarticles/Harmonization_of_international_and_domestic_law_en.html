Ending Execution of Programs or Scripts

In the Command Line or Terminal

  1. Ctrl + C: In most operating systems, pressing the `Ctrl` and `C` keys simultaneously will interrupt the current running process.
  2. kill command: In Unix/Linux systems, you can use the `kill` command to end a process. For example, `kill -9 ` will forcefully terminate the specified process ID.
  3. taskkill command: In Windows systems, you can use the `taskkill` command to end a process. For example, `taskkill /F /PID ` will forcefully terminate the specified process ID.

In Scripts

Python

In a Python script, you can use `sys.exit()` or `exit()` to end the program execution.

import sys
sys.exit("Program ended")

or

exit("Program ended")

Bash

In a Bash script, you can use the `exit` command to end the script execution.

#!/bin/bash
echo "Script ended"
exit 0

PowerShell

In a PowerShell script, you can use the `exit` command to end the script execution.

Write-Output "Script ended"
exit

In the Graphical User Interface (GUI)

  1. Task Manager: In Windows systems, you can open the Task Manager, find the process you want to end, and then click "End Task".
  2. Activity Monitor: In macOS systems, you can open the Activity Monitor, find the process you want to end, and then click "Force Quit".

In Integrated Development Environments (IDEs)

Most IDEs provide a stop button that you can click directly to end the program execution.
