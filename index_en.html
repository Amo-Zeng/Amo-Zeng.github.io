Natural programming language is a Chinese programming language with syntax resembling natural language. It can be seen as a Lisp language without parentheses, as parameters are separated not by parentheses but by spaces.

Here are some examples:

  • Calculate 3 + 4 × 5 - 6: subtract add 3 multiply 4 5 6
  • Read file contents into a variable: read file_name variable_name
  • Write variable contents into a file: write variable_name file_name
  • Ask a question from a large language model and assign it to a variable: let variable_name llmquery question
  • Use a large language model to generate a new webpage: write llmquery "generate an article about the principle of LLM, return in HTML format, return code only." llm.html
